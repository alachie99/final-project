A Cyberbullying Detection System for Chat Application

This is a cyberbullying detection model built on Logistics Regression.

About this System

This is a cyberbullying detection system that seeks to prevent cyberbullying of chat applications before its occurrence.

This system detects harshness in messages before it can be sent to the receiver.

This system can be incorporated into the build of your chat application or chat rooms to minimize and prevent cyeberbullying

System Build

Logistics Regression

This detection System was built on Logistics regression for easy probabilities of detection

Accurate

This model has an 80% accuracy level.

Datasets

Datasets used in this model cover a large range of harsh words.

API

An API for interfacing with the model was built so calls can be made to the model
